# Part 1-2 데이터셋 설명

## 개요
제조 결함 이미지 분류를 위한 비전 데이터. 정상/결함 이미지 분류 및 CNN 기반 품질 검사 실습.

## 포함 샘플 파일

### defect_images/ 디렉터리
| 폴더 | 포함 수 | 원본 전체 수 | 이미지 크기 | 설명 |
|------|---------|-------------|------------|------|
| `normal/` | 3장 | 100+ | 224×224 px | 정상 제품 표면 |
| `scratch/` | 3장 | 100+ | 224×224 px | 스크래치 결함 |
| `contamination/` | 3장 | 100+ | 224×224 px | 오염 결함 |
| `damage/` | 3장 | 100+ | 224×224 px | 파손 결함 |
| `sample_*.png` | 3장 | - | 224×224 px | 혼합 샘플 (데모용) |

**총 포함**: 15장 (Git 저장소 용량 관리를 위해 카테고리당 3장만 포함)

## KAMP 원본 데이터셋 (전체)

**출처**: [KAMP (Korea AI Manufacturing Platform)](https://www.kamp-ai.kr/)

### 제조 결함 이미지 데이터
- **데이터셋명**: 금속 표면 결함 검출 데이터
- **전체 규모**: ~4,000장, ~2GB
- **클래스**: normal, scratch, contamination, damage, dent, weld 등 6+ 클래스
- **이미지 사양**: 224×224~512×512 px, RGB, PNG/JPG
- **레이블링**: 폴더 기반 분류 (폴더명 = 클래스명)

### 데이터 구조
```
defect_images/
├── normal/          # 정상 표면 (~800장)
├── scratch/         # 스크래치 결함 (~600장)
├── contamination/   # 오염 결함 (~500장)
├── damage/          # 파손 결함 (~500장)
├── dent/            # 찌그러짐 (~400장)
└── weld/            # 용접 결함 (~400장)
```

### 다운로드 방법
1. https://www.kamp-ai.kr/ 접속
2. 금속 표면 결함 검출 데이터셋 신청
3. 다운로드 후 `../../dataset/part1-2/defect_images/` 경로에 배치
4. 또는 노트북 실행 시 샘플 데이터로 자동 대체

> 노트북은 KAMP 실데이터가 없으면 자동으로 이 샘플 파일(15장)을 사용합니다.
